# Datenschutzerklärung / Privacy Policy

**Instagram Reels Controls**
Version 1.1.0
Stand: 10. Januar 2026

---

## Deutsch

### Welche Daten werden erfasst?

Die Extension **Instagram Reels Controls** erfasst und speichert folgende Daten **ausschließlich lokal** in Ihrem Browser:

1. **Lautstärke-Einstellung** – Ihre bevorzugte Lautstärke für Videos
2. **Sprungweite** – Ihre gewählte Zeitspanne für Vor-/Zurückspulen (z.B. 5 Sekunden)
3. **Lautstärke-Schritte** – Ihre gewählte Schrittgröße für Lautstärkeänderungen (z.B. 10%)
4. **Auto-Unmute-Einstellung** – Ob Videos automatisch entmutet werden sollen

### Wo werden die Daten gespeichert?

Alle Daten werden **ausschließlich lokal** im Chrome/Firefox-Speicher (`chrome.storage.local` / `browser.storage.local`) gespeichert. Diese Daten:

- Verlassen niemals Ihren Browser
- Werden nicht an externe Server übertragen
- Werden nicht mit Dritten geteilt
- Sind nur für diese Extension zugänglich

### Werden Daten an Server gesendet?

**Nein.** Diese Extension:

- Sendet **keine Daten** an externe Server
- Nutzt **keine Analytics oder Tracking**
- Führt **keine Netzwerkanfragen** durch (außer zum Laden von Instagram selbst)
- Sammelt **keine persönlichen Informationen**

### Welche Berechtigungen werden benötigt?

Die Extension benötigt folgende Berechtigungen:

| Berechtigung | Zweck | Daten-Zugriff |
|--------------|-------|---------------|
| `storage` | Speichern Ihrer Einstellungen lokal | Nur lokale Einstellungen |
| `https://www.instagram.com/*` | Funktioniert nur auf Instagram | Kein Daten-Zugriff, nur UI-Manipulation |

### Datenlöschung

Um alle gespeicherten Daten zu löschen:

1. Öffnen Sie `chrome://extensions/` (oder `about:addons` in Firefox)
2. Klicken Sie auf "Entfernen" bei Instagram Reels Controls
3. Alle lokal gespeicherten Daten werden automatisch gelöscht

Alternativ können Sie die Extension-Daten zurücksetzen:
- Rechtsklick auf Extension-Icon → "Optionen" → Browser-Speicher löschen

### Änderungen dieser Datenschutzerklärung

Bei wesentlichen Änderungen dieser Datenschutzerklärung werden wir die Versionsnummer und das Datum aktualisieren. Die jeweils aktuelle Version finden Sie im Extension-Repository.

### Kontakt

Bei Fragen zum Datenschutz können Sie ein Issue im GitHub-Repository erstellen.

## English

### What data is collected?

The extension **Instagram Reels Controls** collects and stores the following data **exclusively locally** in your browser:

1. **Volume Setting** – Your preferred volume for videos
2. **Seek Step** – Your chosen time interval for seeking (e.g., 5 seconds)
3. **Volume Step** – Your chosen step size for volume changes (e.g., 10%)
4. **Auto-Unmute Setting** – Whether videos should be automatically unmuted

### Where is the data stored?

All data is stored **exclusively locally** in the Chrome/Firefox storage (`chrome.storage.local` / `browser.storage.local`). This data:

- Never leaves your browser
- Is not transmitted to external servers
- Is not shared with third parties
- Is only accessible to this extension

### Is data sent to servers?

**No.** This extension:

- Sends **no data** to external servers
- Uses **no analytics or tracking**
- Makes **no network requests** (except loading Instagram itself)
- Collects **no personal information**

### What permissions are required?

The extension requires the following permissions:

| Permission | Purpose | Data Access |
|------------|---------|-------------|
| `storage` | Store your settings locally | Local settings only |
| `https://www.instagram.com/*` | Works only on Instagram | No data access, UI manipulation only |

### Data Deletion

To delete all stored data:

1. Open `chrome://extensions/` (or `about:addons` in Firefox)
2. Click "Remove" on Instagram Reels Controls
3. All locally stored data will be automatically deleted

Alternatively, you can reset extension data:
- Right-click extension icon → "Options" → Clear browser storage

### Changes to this Privacy Policy

For significant changes to this privacy policy, we will update the version number and date. The current version can always be found in the extension repository.

### Contact

For privacy questions, please create an issue in the GitHub repository.
